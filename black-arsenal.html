Contexte du Projet Académique

🏫 Cadre Pédagogique

Projet de développement web avancé dans le cadre du BUT Informatique à l'IUT Montpellier-Sète. Objectif : maîtriser le développement PHP orienté objet avec architecture MVC et intégration base de données relationnelle.

🎮 Thématique Gaming

Choix créatif d'un e-commerce d'armes virtuelles inspiré de l'univers des jeux vidéo (CS:GO, Valorant, Call of Duty). Approche ludique pour rendre le projet engageant tout en respectant les contraintes techniques académiques.

⚡ Contraintes Techniques

Développement 100% PHP vanilla avec architecture MVC stricte. Base de données MySQL avec requêtes optimisées. Sécurisation complète contre les vulnérabilités web courantes. Interface responsive et accessible.

👥 Collaboration Équipe

Projet réalisé en binôme avec répartition des responsabilités : développement backend/frontend, conception base de données, sécurisation et tests. Utilisation de Git pour le versioning collaboratif.

⚙️

Architecture Technique & Stack

🐘

PHP 8.1

Développement backend 100% PHP vanilla avec programmation orientée objet, gestion des sessions, validation des données et traitement des formulaires sécurisés.

🗄️

MySQL 8.0

Base de données relationnelle avec 8 tables optimisées, requêtes SQL complexes, index de performance et procédures stockées pour la logique métier critique.

🏗️

Architecture MVC

Implémentation stricte du pattern Modèle-Vue-Contrôleur avec séparation claire des responsabilités, routage personnalisé et injection de dépendances.

🎮

Interface Gaming

Frontend immersif avec thème gaming, animations CSS3, responsive design et expérience utilisateur optimisée pour l'e-commerce de gaming.

🏗️ Architecture MVC Implementée

📊 Modèle (Model)

Classes PHP pour la logique métier : User, Product, Order, Cart. Interaction avec la base de données MySQL via PDO. Validation des données et règles business.

👁️ Vue (View)

Templates PHP avec intégration HTML/CSS. Interface gaming responsive, gestion des états d'affichage et présentation des données utilisateur.

🎛️ Contrôleur (Controller)

Routage des requêtes HTTP, orchestration Modèle-Vue, gestion des sessions utilisateur et traitement des actions CRUD sur les produits.

🚀

Fonctionnalités Développées

👤

Système d'Authentification

Inscription, connexion et gestion de profil avec validation des données, hachage sécurisé des mots de passe (password_hash) et gestion des sessions PHP. Système de rôles utilisateur/admin.

PHP Sessions Password Hash Data Validation
🛒

Panier & Commandes

Système de panier persistant en session, gestion des quantités, calcul automatique des totaux avec TVA. Processus de commande complet avec historique et statuts de livraison.

Session Management MySQL Transactions Business Logic
⚔️

Catalogue d'Armes

Base de données complète d'armes gaming avec catégories (pistolets, fusils, sniper), caractéristiques techniques, prix et système de recherche/filtrage avancé.

MySQL Indexes Search Algorithm Category System
🛡️

Administration

Panel d'administration complet pour la gestion des produits (CRUD), utilisateurs, commandes et statistiques. Interface dédiée avec contrôle d'accès par rôles.

Role Management CRUD Operations Admin Dashboard
🔍

Recherche Avancée

Moteur de recherche avec filtres multiples : catégorie, prix, rareté, jeu d'origine. Recherche textuelle optimisée avec suggestions automatiques et tri personnalisable.

Full-text Search Dynamic Filtering AJAX Requests
📊

Système de Stats

Dashboard avec statistiques de vente, produits populaires, revenus par période. Génération de rapports PDF et export CSV pour l'analyse business.

Data Analysis Report Generation Chart.js

Sécurité & Base de Données

🚫

Protection SQL Injection

🚧

Protection XSS

🔑

Gestion Sessions

Sessions sécurisées avec regeneration d'ID et expiration automatique

✅

Validation Données

Contrôle Accès
